# Постановка задачи

## Общее

Пишем маленького агента по теме обучения в вузе (студенты, предметы, успеваемость и тд).
Можно сказать, что студенты уже прошли экзамены и находятся на каникулах.

Агент должен уметь отвечать на некоторые типовые вопросы. Проверка осуществляется через набор end-to-end тестов.

## Глобальная цель

Написать функцию agent, которая будет принимать на вход запрос пользователя и возвращать ответ агента в виде строки (+ экстра инфо).


## Что дано

Есть на руках и нужно использовать:

- данные
- апи к ним
- тесты

Предполагается, что агент сможет извлекать интент/параметры из запроса пользователя и запрашивать необходимые данные через существующие апи.

### API

Представлены в виде python функций в директории src/api.

#### 1. get_students

- на вход название предмета и топ-k (дефолт k=3)
- возвращает информацию топ-k студентов, отсортированных по баллу
- пример запроса: “топ студентов по мл”
- пример ответа: [ {“name”: “John”, “score”: 4.9}, {“name”: “Marie”, “score”: 4.7}, {“name”: “William”, “score”: 3.8} ]
- !! если название предмета пусто / нет в данных, то возвращает пустой список
- !! если значение k < 1, то возвращает пустой список
- !! если значение k > 10, то падает с ошибкой

#### 2. get_avg_score

- на вход название предмета
- возвращает средний балл студентов по указанному предмету
- пример запроса: “средний балл по мл”
- пример ответа: {“avg_score”: 4.4}
- !! если название предмета пусто / нет в данных, то возвращает пустой словарь

#### 3. get_avg_overall_score

- на вход ничего
- возвращает средний балл студентов по всем предметам
- пример запроса: “средний балл студентов”
- пример ответа: {“avg_score”: 4.8}

#### 4. search_rag

- на вход строка-запрос и топ-k (дефолт k=2)
- возвращает набор чанков
- пример запроса: “лектор по мл”
- пример ответа: { “chunks”:  [ “text1”, “text2” ] }

## Что нужно реализовать

### Агент

В директории src есть agent.py с функцией agent - ее нужно написать. В ней зашита вся механика с вызовом API, принятием решений и тд.

Пример запроса пользователя:

```python
user_request = "топ студентов по мл"
```

Пример ответа функции agent:

```python
agent_result = {
    "answer": "...",
    "token_usage": {
        "prompt_tokens": 18,
        "completion_tokens": 24,
        "total_tokens": 42
    }
}
```

Добавление token_usage пока опционально.
