# Песочница ИФТ

На ИФТ доступна песочница с поднятым JupyterLab.
Дополнительно можно запускать код в терминале (ОС SberLinux).

URL: `tslds-efs002401.delta.sbrf.ru:8443`

## Как получить доступ

1. Создать заявку в Сбердруге: `Доступ к стендам разработки и тестирования`.
  - Стенд: `Рабочее место ВСП ЮЛ, DEV (разработка) (CI02407739)`.
  - Обоснование: произвольная формулировка.
4. После выполнения заявки перейти по URL и зарегистрироваться, создав личный логин-пароль.
  Эти учетные данные действуют только на этом стенде, поэтому пароль можно сделать попроще.

## Комментарии

- В корне песочницы есть `README` с подробным описанием доступа к папкам и общими правилами работы.
- Стенд может быть упасть в любой момент, поэтому важные артефакты рекомендуется коммитить в Bitbucket.
